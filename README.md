# VR-Study-Pretest
This repository contains a fork of the Tapping PROMS

# Requirements

- Anaconda or Miniconda
- A pair of headphones

# Create a conda environment

```bash
conda env create -f environment.yml 
```

# Start the program

```bash
conda activate psychopy
python main_test.py
```
